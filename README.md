# city-report-qa Skill

> 城市级品类进入可行性分析报告质检Skill - 8项必含指标 + P0/P1/P2三级硬伤检查

## 🎯 一句话介绍

帮你快速生成199-299元级的城市级品类进入可行性分析报告，并自动校验质量。

## 📊 适用场景

- 连锁品牌跨城扩张：XX品类XX城市能不能进？
- 选址咨询机构：标准化产品交付
- 投资人/FA：行业进入机会研判

## 🛠️ 核心功能

- ✅ 8项必含指标自动检查（饱和度/密度/价格带/评分/品牌/品类/集中度等）
- ✅ P0/P1/P2三级硬伤检查（公式错误/口径不统一/品类边界等）
- ✅ 报告模板（10章标准结构）
- ✅ 采集脚本（高德POI + 24个广义关键词）

## 📦 快速使用

```bash
# 1. 克隆仓库
git clone https://github.com/ysw-2025/city-report-qa.git
cd city-report-qa

# 2. 安装依赖（仅需requests）
pip install requests

# 3. 运行质检工具
python qa_check.py <报告文件路径>
```

## 📋 质检输出

```
【综合评分】
  基础指标: 11/11
  P0级: 3/3
  P1级: 2/2
  P2级: 6/6
  综合: 22/22 (100%)
  评级: ✅ 可交付（90分+）
```

## 🏗️ 8项必含指标

1. 全市门店总量
2. 每万人门店数（饱和度）
3. 各区门店密度
4. 价格带分布
5. 评分分布
6. 头部品牌TOP20
7. 细分品类占比
8. 市场集中度CR5/CR10

## ⚠️ P0级硬伤（必须规避）

- P0-1：回本测算公式（餐位数=面积÷2㎡）
- P0-2：数据口径统一
- P0-3：数据来源标注
